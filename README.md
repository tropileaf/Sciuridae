# Sciuridae
Supplemental

 Contains .fas alignments and subsequent ASAP, ABGD, IQtree and mPTP analyses performed on APOB and CYTB fragments of Sciurids collected across the Congo Basin in the 21st century

 mPTP analyses done using a .newick IQtree input file
 
 See https://spartexplorer.mnhn.fr/delimitation and http://iqtree.cibiv.univie.ac.at/ as well as https://mcmc-mptp.h-its.org
